프로그램 서술 기법
- 의사코드 : 우리 언어로 컴퓨터 프로그램이나 알고리즘이 수행해야 할 내용을 간략히 서술해놓은 것
   ex) text file에서 바꿀 단어, 바뀔 단어 입력받아 바꿔주는 프로그램
	바꿀 단어 입력
	바뀔 단어 입력
	파일 열기
	각 행에 대해 단어 찾기
		각 글자 읽기
		바꾸고 싶은 단어 첫 글자와 일치하면
			일치하는지 확인
			일치하면 단어 찾은 것
		단어 몇번 나왔는지 세기
		그 단어에 해당하는 글자 지우기
		새로운 글자 채워넣기
		단어 몇 번 바꿨는지 사용자에게 알려주기
	파일 닫기

- 순서도 : 일을 처리할 때, 여러 종류의 상자와 화살표를 통해 명령 순서를 보여주는 알고리즘/프로세스
